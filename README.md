# 🎮 Valorant Agent Recommender

A Python CLI tool that recommends the best agents based on:

- Map selection
- Playstyle (Chill / Normal / Tryhard)
- Role preference

## Features
- Smart scoring system
- Map-based agent strengths
- Difficulty-based recommendations
- Clean terminal UI

## How to Run ?

```bash
python main.py
```

Get top 3 agent recommendations based on your inputs.
